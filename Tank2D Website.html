Tank2D

by: Arush Shah

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game, Tank2D because it just crossed my mind in the process of thinking for a game idea. It was first a maze game, then I thought it would be cool if it was two-player and there were walls and you can shoot, and eventually it became a PVP.

2. What I Learned

I learned many things from people testing out my game. People gave me ideas and things to do to improve the game. These are the things that were given to me to improve my game: - Different controls - Health Bar instead of a number of lives - Different amount of points per shot

3. Architecture of My Game

There are 6 classes in the game. The gameplay class is the class that has the scores, the movements, the health bar, and it calls all other classes. It is the most important class. Then there is the brick class. The brick class creates the breakable brown bricks and the indestructible white bricks and includes the code in it. The player1bullet and the player2bullet classes include the bullets. It includes how the bullet functions and what happens when the bullet hits a player. The final class is the main class which makes the frame and sets the text of the play again screen.

4. Technical Implementation

One part of the game that I am proud of is the health bar. Instead, like most games, listing the number of loves left, I used colors to indicate the number. If the bar is green, it meant you have 3 lives left, yellow is 2, red is 1, and black is 0 lives left. I did this by drawing rectangles over each other whenever the lives variable went down. the rectangles all had specific colors.

5. Most Challenging Part

The most challenging part of this project was using the images. I used images from the internet and they just would not show up when run. The code works but the images did not. I fixed the problem by changing the way I created the photos on the game.

6. Future Fixes and Updates

In the future, I would improve the gamification aspect of Tank2D. I would make more features like powerups and maybe even 3 players. All in all, I would add extra features of the game to make it more fun and interesting.
